How to use it?

CLIENT ROUTE

```sh
GET /api/clients
POST /api/clients
PUT /clients/update/:id
DELETE /clients/:id
```

MOVIE ROUTE
```sh
GET /api/movie
POST /api/movie
PUT /movie/update/:id
DELETE /movie/:id
```

RENTAL ROUTE
```sh
GET /api/rental
POST /api/rental
```

USER ROUTE
```sh
GET /api/user
POST /api/user
DELETE /api/user/:id
```
