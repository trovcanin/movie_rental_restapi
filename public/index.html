Welcome Movie Rental REST API


How to use it?

CLIENT ROUTE

GET /api/clients

POST /api/clients

MOVIE ROUTE

GET /api/movie

GET /api/rental

POST /api/rental


USER ROUTE

GET /api/user

POST /api/user

DELETE /api/user/:id
